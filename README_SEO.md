# Google'da Görünürlük İçin SEO Rehberi

## Yapılan İyileştirmeler

### 1. Schema.org Structured Data (JSON-LD)
- ✅ Person schema (Kişi bilgileri)
- ✅ Author schema (Yazar bilgileri)
- ✅ Book schema (Kitap bilgileri)
- ✅ WebSite schema (Site bilgileri)

### 2. Meta Tags
- ✅ Open Graph tags (Facebook, LinkedIn)
- ✅ Twitter Card tags
- ✅ SEO meta tags (description, keywords)
- ✅ Canonical URL

### 3. Dosyalar
- ✅ sitemap.xml (Site haritası)
- ✅ robots.txt (Arama motoru yönlendirmesi)

## Google'da Görünmek İçin Yapılması Gerekenler

### 1. Domain ve Hosting
- Web sitenizi bir domain'e bağlayın (örn: www.kadirveral.com)
- Hosting sağlayıcınızda SSL sertifikası olmalı (HTTPS)

### 2. Google Search Console
1. https://search.google.com/search-console adresine gidin
2. Sitenizi ekleyin
3. Sitemap'i gönderin: `https://www.kadirveral.com/sitemap.xml`
4. URL Inspection ile sayfaları test edin

### 3. Google My Business (Opsiyonel ama Önerilir)
1. https://www.google.com/business/ adresine gidin
2. "Yazar" veya "Sanatçı" kategorisinde profil oluşturun
3. Fotoğraf, biyografi ve iletişim bilgilerini ekleyin

### 4. Sosyal Medya Profilleri
- Facebook sayfası oluşturun
- Instagram profilini optimize edin
- LinkedIn profil oluşturun
- Tüm profillerde aynı bilgileri kullanın

### 5. Backlink Oluşturma
- Edebiyat sitelerinde yazılar yazın
- Kitap inceleme sitelerinde profil oluşturun
- Basın bültenleri gönderin
- Diğer yazarlarla iş birliği yapın

### 6. İçerik Güncellemeleri
- Düzenli blog yazıları yazın
- Kitap tanıtımları yapın
- Etkinlik haberleri paylaşın
- Okuyucu yorumları ekleyin

## Önemli Notlar

### URL'leri Güncelleme
`sitemap.xml` ve `index.html` dosyalarında `https://www.kadirveral.com` yazan yerleri kendi domain'inizle değiştirin.

### Fotoğraf URL'leri
Open Graph ve Twitter Card meta tag'lerinde fotoğraf URL'lerini mutlak URL olarak güncelleyin:
- `https://www.kadirveral.com/public/2.webp`

### Schema.org Verileri
`index.html` içindeki JSON-LD script'lerinde:
- URL'leri kendi domain'inizle değiştirin
- Sosyal medya linklerini güncelleyin
- Fotoğraf URL'lerini mutlak URL yapın
